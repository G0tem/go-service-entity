# go-service-entity

Сервис в разработке и постоянно модифицируется, предназначен для изучения и тестирования инструкментов.  

В сервисе реализован http и gRPC-клиент.  
Реализованы CRUD запросы.    

Тестовый эндпоинт c gRPC  
```bash
# Получите JWT токен через auth сервис
# Затем вызовите тестовый эндпоинт:

curl -X GET http://localhost:8010/api/v1/entity/test_grpc \
  -H "Authorization: Bearer YOUR_JWT_TOKEN"
```

CI/CD настроен на тестовый сервер.
Доступ к документации (если внешний аpi до сих пор оплачен :) доступен по ```http://176.123.164.164:8080/api/v1/docs```  
Запросы идут через Traefik
